AC SOJA

Análisis histórico del cultivo de soja

Herramienta interactiva para explorar el comportamiento observado del cultivo de soja en lotes comerciales de la red CREA COR.

Permite analizar variables ambientales y de manejo, comparar alternativas y estudiar escenarios construidos a partir de información histórica.

Los resultados describen comportamientos observados y no constituyen predicciones ni recomendaciones agronómicas.

¿Qué querés analizar?

Cada módulo responde una pregunta distinta sobre la misma base histórica.

🔍 Explorar

¿Cómo se comportó históricamente el cultivo?

Explorá la relación entre el rendimiento y las variables ambientales, de manejo y productivas registradas en la base histórica.

Explorar datos → ⚖️ Comparar

¿Qué diferencias se observaron entre distintas alternativas?

Compará materiales, fechas de siembra, ciclos, ambientes y otras categorías utilizando la misma base de información histórica.

Comparar resultados → 🧭 Escenarios

¿Cómo se comportaron distintas estrategias de manejo?

Analizá combinaciones de ambiente, fecha de siembra y ciclo para conocer el comportamiento histórico de distintas estrategias de manejo.
